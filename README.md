# SkiaFly
练习Skia

## 项目结构
- include
  存放第三方库的头文件
  
- libs
  存放动态链接库
  
- scripts
  存放各种编译脚本
  
- src
  存放源码
  
- third_party
  存放第三方库源码
  
- resources
  存放资源文件
